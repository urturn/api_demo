How to setup a DbClick campaign ?

#1 - Go to studio.doubleclick.com and login :)

#2 - click 'New Creative'

#3 - select From Layout

#4 - set up your campaign info and select User defined size of 300 x 400 then click continu

#5 - click on 'File Layout' (hidden inside the block of text)

#6 - Drag your layout file (generated by wizzard) and a 300x400 placeholeder image in to the drag file here area. When all upload click continu.

#7 - Drag your layout file (generated by wizzard) and a 300x400 placeholeder image in to the drag file here area

#8 - Click Add exit event then continu

#9 - Preview your campaign then continu

#10 - Wrote a good message to QA and let your campaign go to review!
